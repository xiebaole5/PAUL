# 天虹紧固件视频脚本生成功能 - 更新文档

## 🎯 更新概述

为天虹紧固件小程序新增**视频脚本生成功能**，支持 5-30秒 不同时长的定制化营销脚本生成。

---

## 📋 功能特性

### 1. 支持的时长
- 5秒 - 极简脚本，适合快速品牌曝光
- 10秒 - 精简脚本，适合社交媒体快速传播
- 15秒 - 标准脚本，适合产品展示
- 20秒 - 详细脚本，完整品牌宣传（默认）
- 25秒 - 深度脚本，展示技术实力
- 30秒 - 完整脚本，全面展示

### 2. 脚本内容
每个脚本包含：
- ✅ **场景描述 (Visuals)** - 详细的视觉内容描述
- ✅ **文案/旁白 (Voiceover/Text)** - 旁白或屏幕文字
- ✅ **音效 (Audio)** - 背景音乐和音效建议
- ✅ **关键元素** - 确保包含的核心信息

### 3. 核心信息突出
- 🎯 公司背景：1987年成立，30余年专业经验
- 🎯 产品特色：高难度、特殊紧固件，定制非标
- 🎯 服务行业：汽车、太阳能、机械设备、电表
- 🎯 技术实力：大规模智能制造、定制研发
- 🎯 品牌标识：红色TNHO商标（T-N-H-O）
- 🎯 行动号召：联系网站 zjthfastener.com

## 🚀 使用方法

### 方式一：通过小程序
1. 打开天虹紧固件小程序
2. 输入产品名称（如：高强度螺栓）
3. 选择"📝 生成脚本"
4. 选择主题和时长
5. 点击"生成脚本"
6. 查看并复制脚本

### 方式二：通过 API
```python
from tools.video_script_generator import generate_fastener_promo_script

# 生成20秒脚本
script = generate_fastener_promo_script.func(
    duration=20,
    product_name="高强度螺栓",
    theme="品质保证"
)

print(script)
```
