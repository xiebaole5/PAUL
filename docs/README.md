## 🚀 快速开始

### 首次部署

1. 阅读 [项目总览](../PROJECT_README.md) 了解项目
2. 按照 [完整部署指南](../DEPLOYMENT_GUIDE.md) 部署后端和小程序
3. 按照 [SSL 证书快速参考](SSL_QUICK_REFERENCE.md) 配置 HTTPS

### 修复现有问题

1. 阅读 [服务器问题修复总结](服务器问题修复总结.md) 了解问题
2. 使用 [quick-fix-nginx.sh](../scripts/quick-fix-nginx.sh) 自动修复
3. 按照 [Cloudflare SSL 配置指南](cloudflare-ssl-配置指南.md) 配置 Cloudflare

### 发布小程序

1. 阅读 [小程序正式发布配置完成总结](小程序正式发布配置完成总结.md) 了解当前配置状态
2. 按照 [小程序正式发布-域名配置](小程序正式发布-域名配置.md) 配置域名
3. 按照 [小程序正式发布-快速测试指南](小程序正式发布-快速测试指南.md) 测试功能
4. 测试通过后，上传小程序代码并提交审核
